# go-cover

![github pages](https://github.com/masakurapa/go-cover/workflows/github%20pages/badge.svg)

## Installing

```sh
$ go get -u github.com/masakurapa/go-cover
```

## Usage

```
$ go test -coverprofile=coverage.out ./...
$ go-cover
$ open coverage.html
```
