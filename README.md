# gover-html

![github pages](https://github.com/masakurapa/gover-html/workflows/github%20pages/badge.svg)

## Installing

```sh
$ go get -u github.com/masakurapa/gover-html
```

## Usage

```
$ go test -coverprofile=coverage.out ./...
$ gover-html
$ open coverage.html
```
